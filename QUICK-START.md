# 快速开始指南

## 🚀 5 分钟上手

### 第 0 步：查看可视化示例（强烈推荐！）

在开始之前，先直观地看看每种风格的实际效果：

```bash
# 打开示例导航页
cd examples
open index.html

# 或直接打开某个风格的示例
open minimalist-example.html
```

**在线查看**：双击 `examples/index.html` 文件，浏览器会打开一个导航页面，展示所有 5 种风格的完整示例。

### 第 1 步：选择风格

浏览以下设计风格，找到最符合你产品定位的：

| 风格 | 适用场景 | 特点 |
|------|---------|------|
| **极简风格** | B2B 工具、专业产品 | 简洁、专注、高效 |
| **现代科技风** | AI 产品、开发工具 | 渐变、3D、科技感 |
| **专业商务风** | 企业级 SaaS、CRM | 深色、数据、信任感 |
| **创意大胆风** | 创意工具、协作平台 | 鲜艳、有趣、活泼 |
| **玻璃态风格** | 云服务、现代应用 | 半透明、优雅、轻盈 |

### 第 2 步：复制提示词模板

打开对应风格的 `.md` 文件（如 `prompts/landing-pages/minimalist.md`），根据需求选择：
- **完整版**：详细的设计要求，适合复杂项目
- **快速版**：精简的要点，适合快速原型

### 第 3 步：定制化

根据你的产品信息，补充以下内容：

```
【你的产品】
- 产品名称：[如 AI 写作助手]
- 目标用户：[如内容创作者、营销人员]
- 核心功能：[列出 3-5 个主要功能]
- 品牌色：[如 #6C5CE7]
- 差异化特点：[如"最懂中文的 AI"]

【粘贴选择的提示词模板】

【补充需求】
- [任何特殊需求]
```

### 第 4 步：与 AI 对话

将完整的提示词发送给：
- Cursor（AI 辅助编码）
- ChatGPT / Claude（设计方案）
- Midjourney（视觉设计）

### 第 5 步：迭代优化

根据生成结果，进一步优化：
- "调整配色为更柔和的色调"
- "简化动画效果"
- "增加社会证明区域"

## 🎯 提示词优化技巧

### 1. 明确目标用户
❌ 不好："设计一个落地页"  
✅ 好："设计一个面向企业 CTO 的落地页"

### 2. 提供具体数字
❌ 不好："有很多用户"  
✅ 好："已有 50,000+ 企业用户"

### 3. 参考具体网站
❌ 不好："现代风格"  
✅ 好："类似 Linear.app 的极简风格"

### 4. 分步骤实施
❌ 不好："一次性要求所有功能"  
✅ 好："先生成布局，再完善交互，最后优化动效"

### 5. 提供视觉参考
✅ 好："主色调用类似 Notion 的米白色背景"

## 🔧 常见问题

### Q: 提示词太长，AI 理解不了怎么办？
A: 使用"快速版"提示词，或分步骤提供需求。

### Q: 生成的设计不符合预期？
A: 提供更具体的参考网站，或使用"类似 XXX 的风格"。

### Q: 多种风格可以混合使用吗？
A: 可以，但建议以一种风格为主，其他风格作为点缀。

### Q: 如何确保品牌一致性？
A: 在提示词中明确品牌色、字体、Logo 使用规范。
