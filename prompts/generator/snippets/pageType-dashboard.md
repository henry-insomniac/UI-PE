请生成一个 Dashboard 仪表板页面，布局为 Sidebar + Topbar + Main Content。需要包含：

1) Sidebar（侧边栏）
- Logo
- 6–8 个导航项（含 active 状态）
- 底部用户区（头像/姓名/菜单）

2) Topbar（顶部栏）
- 页面标题或面包屑
- 搜索框（可带 ⌘K 提示）
- 通知图标 + 徽标
- 用户菜单

3) Main Content
- Stats Cards：4 个 KPI 卡片（数字使用等宽字体；趋势用语义色）
- Charts：至少 2 个（折线趋势 + 结构占比/柱状对比）
- Table：最近活动/订单/用户表格（含状态 badge、数值对齐、hover 行）
- Filters：表格上方的筛选/日期范围/搜索（至少 2 个控件）

【信息密度原则】
- Dashboard 的高级感来自“层级 + 对齐 + 字体 + 边框策略”，不是装饰
- 卡片数量克制：不要把所有信息都塞进卡片
