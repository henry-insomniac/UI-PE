# 渐变色风格 (Gradients) - SaaS 落地页提示词

> 📌 来源：基于 [saaspo.com](https://saaspo.com/page-types/saas-landing-page-examples) 的 Gradients 分类

## 🎨 风格特征

- 大胆使用渐变色作为视觉焦点
- 可用于背景、文字、按钮、装饰元素
- 现代、动态、视觉冲击力强
- 区分于单一纯色的设计
- 营造深度和层次感
- 流行于科技和创新产品

- Stripe
- Linear
- Vercel
- Figma (品牌页面)
- Webflow
- Framer

## 🤖 完整提示词模板

```
请设计一个渐变色风格 (Gradients) 的 SaaS 产品落地页，具体要求：

【整体风格】
- 大胆使用渐变色作为核心视觉元素
- 可以是背景渐变、文字渐变、或元素渐变
- 现代、大胆、视觉冲击力强
- 营造动态感和科技感

【渐变类型选择】

1. **线性渐变 (Linear Gradient)**
   - 方向：从左到右、从上到下、对角线
   - 适合：背景、大面积元素
   - 示例：background: linear-gradient(135deg, #667eea, #764ba2);

2. **径向渐变 (Radial Gradient)**
   - 从中心向外扩散
   - 适合：聚焦效果、光晕
   - 示例：background: radial-gradient(circle, #667eea, #764ba2);

3. **多色渐变 (Multi-color)**
   - 3 种或更多颜色过渡
   - 适合：活力、创意感
   - 示例：linear-gradient(90deg, #ff6b6b, #feca57, #48dbfb);

4. **网格渐变 (Mesh Gradient)**
   - 多个渐变点形成复杂的色彩混合
   - 适合：高端、艺术感设计
   
【推荐渐变配色方案】

**科技蓝紫**（最流行）
- 颜色：#667eea → #764ba2
- 感觉：科技、专业、创新
- 适合：AI 产品、开发工具

**日落橙粉**
- 颜色：#ff6b6b → #feca57
- 感觉：温暖、活力、热情
- 适合：社交应用、创意工具

**海洋青蓝**
- 颜色：#00d4ff → #0099ff
- 感觉：清新、可信、科技
- 适合：云服务、SaaS 工具

**极光绿紫**
- 颜色：#11998e → #38ef7d
- 感觉：自然、环保、创新
- 适合：健康科技、绿色产品

**霓虹多彩**
- 颜色：#ff00ff → #00ffff → #ffff00
- 感觉：大胆、前卫、年轻
- 适合：创意平台、娱乐产品

【渐变应用场景】

1. **背景渐变**
   - 全屏渐变背景
   - 渐变叠加在图片上
   - 渐变配合模糊效果
   
2. **文字渐变**
   - 大标题使用渐变文字
   - 代码：
     background: linear-gradient(...);
     -webkit-background-clip: text;
     -webkit-text-fill-color: transparent;

3. **按钮渐变**
   - CTA 按钮使用渐变背景
   - 悬停时渐变方向变化或增强

4. **边框渐变**
   - 卡片边框使用渐变
   - 使用 border-image 或伪元素实现

5. **装饰元素**
   - 渐变的圆形/椭圆形光晕
   - 作为背景装饰

【视觉效果】
- 渐变动画：背景渐变缓慢移动
  animation: gradient-shift 10s ease infinite;
  
- 悬停变化：渐变方向或颜色变化
  
- 光晕效果：渐变圆形 + blur 创造光斑

【与其他元素搭配】
- 深色背景 + 渐变文字/按钮
- 白色背景 + 渐变装饰元素
- 渐变背景 + 白色/深色内容区

【排版建议】
- 在渐变背景上使用白色或深色文字
- 确保足够的对比度
- 渐变文字只用于标题，正文保持纯色

【响应式考虑】
- 背景渐变在不同屏幕上表现一致
- 移动端可以简化渐变效果
```

## 💡 使用技巧

1. **不要过度使用**：渐变是点缀，不是全部覆盖
2. **保持可读性**：渐变背景上的文字要清晰
3. **品牌一致**：渐变色应该与品牌色协调
4. **微妙过渡**：颜色过渡要平滑自然
5. **性能考虑**：避免过于复杂的动态渐变

## 🔧 代码片段

```css
/* 渐变文字 */
.gradient-text {
  background: linear-gradient(135deg, #667eea, #764ba2);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

/* 渐变按钮 */
.gradient-button {
  background: linear-gradient(135deg, #667eea, #764ba2);
  border: none;
  color: white;
  transition: all 0.3s;
}

.gradient-button:hover {
  background: linear-gradient(135deg, #764ba2, #667eea);
  transform: translateY(-2px);
}

/* 渐变边框 */
.gradient-border {
  position: relative;
  background: white;
  border-radius: 16px;
}

.gradient-border::before {
  content: '';
  position: absolute;
  inset: -2px;
  background: linear-gradient(135deg, #667eea, #764ba2);
  border-radius: 18px;
  z-index: -1;
}

/* 动态渐变背景 */
@keyframes gradient-shift {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

.animated-gradient {
  background: linear-gradient(270deg, #667eea, #764ba2, #667eea);
  background-size: 200% 200%;
  animation: gradient-shift 10s ease infinite;
}
```
