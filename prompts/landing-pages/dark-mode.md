# 深色模式 (Dark Mode) - SaaS 落地页提示词

> 📌 来源：基于 [saaspo.com](https://saaspo.com/page-types/saas-landing-page-examples) 的 Dark Mode 分类

## 🎨 风格特征

- 深色背景（纯黑、深灰、深蓝）
- 亮色文字形成高对比度
- 减少眼部疲劳，适合长时间浏览
- 现代、酷炫、专业的视觉感受
- 突出发光效果和亮色元素
- 适合展示产品界面截图

## 📸 参考案例（saaspo.com 上的典型案例）

- Linear
- Vercel
- Raycast
- Warp
- Fig
- Railway

## 🤖 完整提示词模板

```
请设计一个深色模式 (Dark Mode) 的 SaaS 产品落地页，具体要求：

【整体风格】
- 采用深色背景为主的设计
- 现代、专业、科技感强
- 高对比度，文字清晰易读
- 适合技术产品和开发者工具

【配色方案】
- 背景色选择（任选其一）：
  - 纯黑：#000000 或 #0a0a0a
  - 深灰：#111111 或 #171717
  - 深蓝：#0f172a 或 #1e293b
  - 渐变深色：从 #0a0a0a 到 #1a1a2e
  
- 文字颜色：
  - 主标题：纯白 #ffffff
  - 正文：浅灰 #e5e5e5 或 #d1d5db
  - 次要文字：灰色 #9ca3af 或 #6b7280
  
- 强调色（任选一个作为主色）：
  - 蓝色系：#3b82f6 / #60a5fa
  - 紫色系：#8b5cf6 / #a78bfa
  - 绿色系：#10b981 / #34d399
  - 渐变：从主色到次色的渐变
  
- CTA 按钮：
  - 使用强调色作为背景
  - 或使用白色按钮形成对比

【布局结构】
- Hero 区域：
  - 居中或左对齐的大标题
  - 标题可以使用渐变色效果
  - 副标题使用浅灰色
  - 主 CTA 按钮醒目突出
  - 可选：产品界面截图（深色界面）
  
- 功能展示：
  - 卡片使用深色（比背景略浅）
  - 边框：1px solid rgba(255,255,255,0.1)
  - 图标使用强调色或白色
  
- 代码/终端展示（如适用）：
  - 代码块使用更深的背景
  - 语法高亮使用亮色

【视觉元素】
- 边框：使用半透明白色边框分隔区域
  - border: 1px solid rgba(255, 255, 255, 0.1)
  
- 阴影：使用柔和的发光效果
  - box-shadow: 0 0 30px rgba(主色, 0.3)
  
- 背景装饰（可选）：
  - 渐变光晕
  - 网格线图案
  - 噪点纹理（subtle noise）
  
- 分隔线：
  - 使用渐变线或半透明线
  - 或使用强调色作为分隔

【排版】
- 字体：无衬线字体（Inter、SF Pro、Geist）
- 标题：加粗，可使用渐变文字效果
- 字号：
  - 主标题：56-72px
  - 副标题：18-22px
  - 正文：16px

【交互效果】
- 按钮悬停：
  - 增加亮度或发光效果
  - 轻微上浮
- 卡片悬停：
  - 边框变亮
  - 添加微弱的光晕
- 链接：使用强调色，悬停时变亮

【注意事项】
- 确保足够的对比度（WCAG 标准）
- 避免纯黑配纯白造成刺眼
- 深色界面更适合展示深色产品截图
- 考虑提供浅色模式切换选项
```

## 🎯 快速版提示词

```
创建深色模式的 SaaS 落地页：
- 深色背景：#0a0a0a 或 #111111
- 文字：白色 #ffffff + 浅灰 #9ca3af
- 强调色：蓝色 #3b82f6（用于 CTA 和高亮）
- 卡片背景：rgba(255,255,255,0.05)
- 边框：1px solid rgba(255,255,255,0.1)
- 标题可使用渐变色效果
- CTA 按钮使用强调色，带轻微发光
- 悬停时增加亮度和光晕
- 适合：开发工具、技术产品、专业 SaaS
```

## 🔧 深色背景方案对比

| 方案 | 色值 | 感觉 | 适用场景 |
|------|------|------|----------|
| 纯黑 | #000000 | 极简、强对比 | 高端产品 |
| 深灰 | #111111 | 柔和、现代 | 通用 |
| 深蓝 | #0f172a | 专业、科技 | 技术产品 |
| 渐变深色 | 多色 | 丰富、动态 | 创新产品 |

## 💡 使用技巧

1. **选择合适的深度**：不一定要纯黑，深灰往往更舒适
2. **对比度检查**：使用工具检查文字对比度是否达标
3. **发光效果**：善用 box-shadow 和 filter 创造光晕
4. **渐变文字**：标题使用渐变色可以增加视觉层次
5. **产品截图**：如果产品本身是深色界面，展示会更协调
