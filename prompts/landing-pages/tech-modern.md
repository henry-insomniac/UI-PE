# 现代科技风 - SaaS 落地页提示词

## 🎨 风格特征

- 渐变色和霓虹配色
- 3D 元素和深度效果
- 动态背景和粒子效果
- 科技感的网格和线条
- 玻璃态或新拟态元素
- 前沿的视觉体验

## 📸 参考案例

- Vercel
- Framer
- OpenAI
- GitHub Copilot
- Warp Terminal

## 🤖 完整提示词模板

```
请设计一个现代科技风格的 SaaS 产品落地页，具体要求：

【整体风格】
- 现代科技感，面向未来的设计语言
- 使用渐变色和动态效果营造科技氛围
- 深色背景为主，突出发光效果
- 强调创新和前沿技术

【配色方案】
- 背景：深色渐变（从深蓝 #0a0e27 到深紫 #1a0b2e）
- 主色调：蓝紫渐变（#667eea → #764ba2）或青蓝渐变（#00d4ff → #0099ff）
- 强调色：霓虹色（青色 #00ffff、紫色 #ff00ff）
- CTA 按钮：发光效果的渐变按钮
- 文字：白色 (#ffffff) 和半透明白色 (rgba(255,255,255,0.7))

【布局结构】
- Hero 区域：
  - 左侧：大标题（带渐变文字效果）+ 描述 + CTA
  - 右侧：产品演示（3D 界面或动态效果）
  - 背景：动态网格或粒子效果
  
- 功能介绍：
  - 使用卡片式布局，带玻璃态效果（glassmorphism）
  - 每个功能配有图标（发光效果）和动画
  - 鼠标悬停时卡片浮起，带光晕

- 技术展示：
  - 代码块展示（带语法高亮）
  - API 接口演示
  - 实时数据可视化动画

- 社会证明：
  - 客户 Logo 带微光效果
  - 数据统计（带数字递增动画）

【视觉效果】
- 背景：
  - 动态渐变背景
  - 网格线（带透视效果）
  - 浮动的粒子或光点
  - 模糊的光晕效果（blur circles）

- 卡片和容器：
  - 玻璃态效果：半透明背景 + 模糊（backdrop-filter: blur）
  - 边框：1px 渐变边框或发光边框
  - 阴影：大范围的彩色阴影（如蓝色或紫色）

- 按钮：
  - 渐变背景
  - 发光效果（box-shadow: 0 0 20px rgba(色值, 0.5)）
  - 悬停时增强光晕

【排版】
- 字体：
  - 标题：几何无衬线字体（如 Space Grotesk、Outfit）
  - 正文：Inter、SF Pro
- 标题效果：
  - 渐变文字（gradient text）
  - 可选：发光效果或描边效果
- 字号：
  - 主标题：64-80px（大胆使用大字号）
  - 副标题：18-20px

【动画效果】
- 滚动视差效果（parallax scrolling）
- 元素渐入时带有弹性动画（spring animation）
- 鼠标跟随效果（cursor follow）
- 数字递增动画（counting animation）
- 悬停时的 3D 倾斜效果（tilt effect）
- 背景渐变缓慢移动（animated gradient）

【3D 元素】
- 3D 产品展示（可旋转）
- 立体的卡片层次
- 透视网格背景
- 浮动的几何形状

【响应式设计】
- 移动端：简化动画效果，保持性能
- 桌面端：充分利用动态效果和 3D 元素
- 暗色模式优先

【技术实现建议】
- CSS：backdrop-filter、linear-gradient、box-shadow
- JavaScript：GSAP、Three.js（3D 效果）、Framer Motion
- 性能：使用 GPU 加速，优化动画性能
```

## 🎯 快速版提示词（精简版）

```
创建现代科技风格的 SaaS 落地页：
- 深色渐变背景（深蓝到深紫）
- 蓝紫或青蓝色渐变主色调
- 玻璃态卡片（半透明 + 模糊背景）
- 发光的 CTA 按钮（带阴影光晕）
- 动态网格或粒子背景
- 渐变文字标题
- 3D 产品展示或界面演示
- 悬停时卡片浮起带光晕
- 滚动视差和弹性动画
- 代码块展示（语法高亮）
- 数字递增动画
```

## 🔧 可定制参数

根据具体需求，修改以下参数：

- **渐变方向**：调整渐变色方向和过渡点
- **发光强度**：根据品牌调整光晕的强度和颜色
- **动画速度**：根据目标用户调整动效的快慢
- **3D 程度**：根据性能要求决定 3D 元素的复杂度
- **技术展示**：如果是开发工具，强化代码展示

## 💡 使用技巧

1. **性能优化**：告诉 AI 需要考虑性能，避免过度动画
2. **渐进增强**：先实现基础版，再添加动态效果
3. **品牌适配**：用你的品牌色替换渐变色方案
4. **移动优先**：确保移动端也有良好体验

```
【你】：我需要设计一个 AI 代码助手的落地页，目标用户是开发者。
希望采用现代科技风格，突出技术感和未来感。

【粘贴上面的完整提示词模板，并补充】：
- 主色调：青蓝渐变（#00d4ff → #0099ff）
- 核心功能：智能代码补全、Bug 检测、代码重构
- 技术展示：实时代码生成演示、支持的编程语言
- 信任元素：GitHub Stars 数量、开发者评价

特别要求：
- 包含代码编辑器的动态演示
- 代码自动补全的动画效果
- 突出 AI 驱动的智能感

【AI 开始生成】...
```

## ⚠️ 注意事项

- **避免过度设计**：动效要有目的，不要为了炫技而炫技
- **性能优先**：确保页面加载速度快，动画流畅
- **可访问性**：提供关闭动画的选项（prefers-reduced-motion）
- **浏览器兼容**：某些效果（如 backdrop-filter）需要降级方案

---

**最佳适用场景**：开发工具、AI 产品、技术平台、面向技术人员的 SaaS、创新型产品
