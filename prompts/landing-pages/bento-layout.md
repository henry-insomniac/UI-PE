# Bento 布局风格 - SaaS 落地页提示词

> 📌 来源：基于 [saaspo.com](https://saaspo.com/page-types/saas-landing-page-examples) 的 Bento 分类

## 🎨 风格特征

- 类似日式便当盒的网格布局
- 不同大小的卡片/模块组合
- 信息分块清晰，层次分明
- 现代、有组织感、视觉丰富
- Apple 风格的功能展示方式
- 适合展示多个功能或特性

## 📸 参考案例

- Apple (功能页面)
- Linear
- Raycast
- Craft
- Arc Browser
- Framer

## 🤖 完整提示词模板

```
请设计一个 Bento 布局风格的 SaaS 产品落地页，具体要求：

【整体风格】
- 采用 Bento Grid（便当盒网格）布局
- 不同大小的卡片模块组合
- 类似 Apple 产品页面的功能展示方式
- 现代、有条理、视觉层次丰富

【布局网格】
- 使用 CSS Grid 实现
- 基础网格：12 列或 6 列
- 卡片可以占据不同的列数和行数
- 常见组合：
  - 2x2 + 1x2 + 1x2（大卡片 + 两个小卡片）
  - 1x1 + 1x1 + 2x1（两个小方块 + 一个宽卡片）
  - 3x2（一个超大的主功能卡片）
- 卡片之间保持一致的间距（16-24px）

【卡片设计】
- 圆角：16-24px（较大的圆角）
- 背景：
  - 浅色系：白色或浅灰 #f5f5f5
  - 深色系：深灰 #1c1c1e 或 #2c2c2e
  - 或使用品牌色作为某些卡片背景
- 内边距：24-40px
- 每个卡片内容：
  - 图标或小插图
  - 功能标题（大号）
  - 简短描述（1-2 行）
  - 可选：动态演示或截图

【配色方案】
- 背景：中性色背景（白色、浅灰、深灰）
- 卡片：
  - 大部分使用中性色
  - 1-2 个重点卡片使用品牌色或渐变
- 文字：
  - 标题：黑色或白色（高对比）
  - 描述：灰色

【Hero 区域】
- 标题 + 副标题 + CTA（传统布局）
- 或者直接用一个超大的 Bento 卡片作为 Hero
- 可以在 Hero 下方直接开始 Bento 网格

【Bento 功能区域示例布局】

布局 1（标准）：
```
┌─────────┬─────────┬─────────┐
│         │    B    │    C    │
│    A    ├─────────┼─────────┤
│ (2x2)   │    D    │    E    │
├─────────┴─────────┴─────────┤
│            F (3x1)          │
└─────────────────────────────┘
```

布局 2（交错）：
```
┌─────────┬───────────────────┐
│    A    │        B          │
│ (1x2)   │      (2x2)        │
├─────────┤                   │
│    C    ├─────────┬─────────┤
│ (1x1)   │    D    │    E    │
└─────────┴─────────┴─────────┘
```

【卡片内容建议】
- 大卡片（2x2）：主要功能 + 动态演示/大插图
- 中卡片（2x1）：次要功能 + 描述 + 图标
- 小卡片（1x1）：单个特性 + 图标

【视觉效果】
- 卡片悬停：
  - 轻微上浮（transform: translateY(-4px)）
  - 增加阴影
  - 或边框高亮
- 卡片内动画：
  - 图标可以有微动效
  - 数据可以有动态变化
- 过渡动画：
  - 页面加载时卡片依次淡入
  - staggered animation

【排版】
- 标题：加粗，24-32px
- 描述：常规，14-16px
- 字体：无衬线字体（SF Pro、Inter）

【响应式设计】
- 桌面端：完整的 Bento 网格
- 平板端：简化为 2 列
- 移动端：单列堆叠
```

## 🎯 快速版提示词

```
创建 Bento 布局风格的 SaaS 落地页：
- 使用 CSS Grid 创建不规则网格布局
- 卡片大小不一：大卡片(2x2)、中卡片(2x1)、小卡片(1x1)
- 卡片间距：16-24px
- 卡片圆角：16-24px
- 每个卡片包含：图标 + 标题 + 简短描述
- 大卡片可包含产品演示或大插图
- 主色调：中性色（白/浅灰）+ 1-2 个彩色重点卡片
- 悬停效果：轻微上浮 + 增加阴影
- 加载动画：卡片依次淡入（staggered）
- 响应式：移动端变为单列
```

## 🔧 布局代码参考

```css
.bento-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: 200px;
  gap: 20px;
}

/* 大卡片 2x2 */
.card-large {
  grid-column: span 2;
  grid-row: span 2;
}

/* 宽卡片 2x1 */
.card-wide {
  grid-column: span 2;
}

/* 高卡片 1x2 */
.card-tall {
  grid-row: span 2;
}

/* 响应式 */
@media (max-width: 768px) {
  .bento-grid {
    grid-template-columns: 1fr;
  }
}
```

## 💡 使用技巧

1. **视觉平衡**：大卡片和小卡片要搭配协调
2. **信息层次**：最重要的功能放在大卡片
3. **色彩点缀**：用 1-2 个彩色卡片打破单调
4. **内容精简**：每个卡片聚焦一个功能点
5. **动态内容**：大卡片可以放产品演示 GIF 或视频

