# 以文案为主风格 (Copy Focused) - SaaS 落地页提示词

> 📌 来源：基于 [saaspo.com](https://saaspo.com/page-types/saas-landing-page-examples) 的 Copy focused 分类

## 🎨 风格特征

- 文字内容是页面的核心
- 视觉设计服务于文案阅读
- 简洁的设计，不分散注意力
- 强调标题和价值主张
- 适合产品价值需要解释的场景
- 转化率导向的设计

## 📸 参考案例

- Basecamp
- Hey.com
- 37signals 系列产品
- Superhuman
- ConvertKit
- Carrd

## 🤖 完整提示词模板

```
请设计一个以文案为核心的 SaaS 产品落地页，具体要求：

【整体风格】
- 文案是页面的主角，设计为文案服务
- 简洁、专注、没有多余的视觉干扰
- 强调可读性和信息传递
- 让用户聚焦于产品价值

【排版原则】
- 大量的留白，让文字呼吸
- 行宽控制在 60-80 字符（最佳阅读体验）
- 行高 1.6-1.8（正文）
- 段落之间足够的间距

【字体选择】
- 衬线字体（更适合长文阅读）：
  - Georgia、Merriweather、Lora、Playfair Display
- 或无衬线字体（现代感）：
  - Inter、Helvetica、System UI
- 建议：标题用无衬线，正文用衬线（或反过来形成对比）

【字号层次】
- 主标题：48-72px，加粗
- 副标题：24-32px
- 正文：18-20px（比一般网页大）
- 小字说明：14-16px

【配色方案】
- 极简配色，通常只有 2-3 种颜色：
  - 背景：白色 #ffffff 或米白色 #faf9f7
  - 正文：深灰/黑色 #1a1a1a 或 #333333
  - 强调：单一品牌色（用于链接和 CTA）
- 避免过多的颜色干扰阅读

【页面结构】

1. **Hero 区域**
   - 强有力的主标题（一句话说明产品价值）
   - 2-3 句的副标题（解释主标题）
   - 单个明确的 CTA

2. **问题陈述**
   - 描述用户面临的问题
   - 使用用户的语言
   - 让用户产生共鸣

3. **解决方案**
   - 产品如何解决这些问题
   - 具体、可信的描述
   - 可以使用编号列表

4. **证明/背书**
   - 用户评价（真实的长评价）
   - 案例故事
   - 数据证明

5. **最终 CTA**
   - 重申价值主张
   - 清晰的行动号召

【文案写作建议】
- 使用"你"而不是"用户"
- 强调利益而不是功能
- 具体而不是抽象
- 简短的句子和段落
- 使用对话式的语气

【视觉元素极简化】
- 图片：只在必要时使用
- 图标：简洁的线性图标
- 装饰：几乎没有
- 动画：极少或没有

【CTA 设计】
- 按钮文案要具体：
  - ✅ "开始 14 天免费试用"
  - ❌ "提交" 或 "开始"
- 按钮颜色：使用品牌强调色
- 按钮大小：足够大，但不夸张

【移动端优化】
- 文字大小在移动端保持可读
- 段落更短
- 更多的留白
```

## 🎯 快速版提示词

```
创建以文案为主的 SaaS 落地页：

【排版】
- 内容区最大宽度：720px（居中）
- 正文字号：18-20px
- 行高：1.7
- 大量留白

【配色】
- 背景：白色或米白 #faf9f7
- 文字：深灰 #1a1a1a
- 强调色：只用于链接和 CTA

【字体】
- 标题：无衬线（Inter 或 Helvetica），加粗
- 正文：衬线（Georgia 或 Merriweather）或无衬线

【结构】
1. Hero：主标题 + 2 句副标题 + CTA
2. 问题描述（2-3 段）
3. 解决方案（编号列表）
4. 用户评价（真实长评价）
5. 最终 CTA

【避免】
- 过多的图片和装饰
- 复杂的动画
- 多种颜色
```

## 💡 文案技巧

### 标题公式

1. **利益 + 时间**
   - "30 天内提升你的销售转化率"

2. **问题 + 解决**
   - "厌倦了混乱的项目管理？"

3. **具体数字**
   - "超过 50,000 个团队使用"

4. **对比**
   - "更少的会议，更多的成果"

### 好的开头段落

```
你有没有这样的经历：每天花 3 小时在邮件上，
却感觉什么都没完成？

[产品名] 帮助你在 30 分钟内处理完一天的邮件，
让你专注于真正重要的工作。
```

## 📝 对话示例

```
【你】：我需要设计一个邮件客户端的落地页，
产品的核心优势是节省用户时间。希望用文案驱动转化。

【使用提示词】：
- 主标题："收件箱零的秘密"
- 副标题：
  "每天花 3 小时在邮件上？
   我们帮你缩短到 30 分钟。"
- 内容结构：
  1. 描述邮件焦虑的痛点
  2. 介绍产品的 3 个核心功能
  3. 展示节省时间的真实数据
  4. 3 条用户长评价
  5. CTA："开始 14 天免费试用"
- 视觉：极简，无多余装饰
- 字体：标题 Inter，正文 Georgia
```

## ⚠️ 注意事项

1. **文案质量是关键**：设计再好，文案差也无效
2. **不要太长**：虽然以文案为主，但也要精炼
3. **移动端测试**：确保在手机上阅读体验良好
4. **A/B 测试**：测试不同的标题和 CTA 文案
5. **适当的图片**：必要时加入产品截图或用户头像

---

**最佳适用场景**：需要解释的产品、高客单价 SaaS、B2B 服务、创始人/个人品牌、强调信任的产品
