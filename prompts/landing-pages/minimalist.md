# 极简风格 - SaaS 落地页提示词

## 🎨 风格特征

- 大量留白，视觉呼吸感强
- 简洁的图标和线条
- 单一或双色配色方案
- 清晰的层次结构
- 聚焦核心信息
- 无干扰的用户体验

## 📸 参考案例

- Linear.app
- Notion
- Stripe
- Vercel

## 🤖 完整提示词模板

```
请设计一个极简风格的 SaaS 产品落地页，具体要求：

【整体风格】
- 采用极简主义设计理念
- 大量留白，给用户视觉呼吸空间
- 聚焦核心价值主张，避免信息过载
- 使用简洁的几何图形和线条

【配色方案】
- 主色调：[白色/浅灰色背景]
- 强调色：[单一品牌色，如蓝色 #0066FF]
- CTA 按钮：[与强调色对比的深色，如黑色 #000000]
- 文字：黑色 (#1a1a1a) 和灰色 (#666666) 层次

【布局结构】
- Hero 区域：居中对齐，大标题 + 副标题 + 单个主 CTA
- 功能介绍：3-4 个核心功能，使用简洁图标 + 标题 + 一句话描述
- 社会证明：客户 Logo 墙或简短评价（灰度处理）
- 底部 CTA：重复行动号召，保持简洁

【排版】
- 字体：无衬线字体（如 Inter、SF Pro、Helvetica）
- 标题：大字号（48-72px），加粗
- 正文：中等字号（16-18px），行高 1.6
- 间距：充足的上下边距（80-120px 区块间距）

【图标与插图】
- 使用线性图标（stroke-based icons）
- 避免复杂的插图，可用简单的几何形状
- 如果需要产品截图，使用干净的界面截图，带有轻微阴影

【交互效果】
- 按钮悬停：轻微的颜色变化或阴影
- 滚动动画：淡入效果（fade-in），不要过于复杂
- 链接：下划线或颜色变化

【响应式设计】
- 移动端：单列布局，保持充足留白
- 桌面端：居中内容，最大宽度 1200px

【避免的元素】
❌ 复杂的背景图案
❌ 多种颜色混用
❌ 过多的动画效果
❌ 冗长的文案
❌ 过于装饰性的元素
```

## 🎯 快速版提示词（精简版）

```
创建极简风格的 SaaS 落地页：
- 白色背景 + 单一品牌色（如蓝色）
- 大标题居中，突出核心价值
- 3-4 个核心功能，线性图标 + 简短描述
- 充足留白（区块间距 100px+）
- 无衬线字体，清晰排版
- CTA 按钮醒目，使用对比色
- 客户 Logo 墙（灰度处理）
- 轻微的悬停和淡入动画
```

## 🔧 可定制参数

根据具体需求，修改以下参数：

- **品牌色**：替换为你的品牌主色
- **功能数量**：根据产品调整为 3-6 个
- **目标用户**：调整文案风格（技术向 / 商务向）
- **行业**：添加行业特定的信任元素

## 💡 使用技巧

1. **先问后设计**：告诉 AI 你的产品是什么，目标用户是谁
2. **分步骤执行**：先设计布局，再细化组件
3. **提供参考**：如果有喜欢的网站，可以说"类似 Linear.app 的风格"
4. **迭代优化**：先生成基础版，再逐步调整细节
