# 创意大胆风 - SaaS 落地页提示词

## 🎨 风格特征

- 鲜艳的配色和大胆的色彩组合
- 不规则的布局和创意排版
- 手绘插画或 3D 角色
- 趣味性和个性化
- 打破常规的设计元素
- 强烈的视觉冲击力

## 📸 参考案例

- Loom
- Figma (品牌页面)
- Mailchimp
- Slack
- Dropbox (早期设计)
- Asana

## 🤖 完整提示词模板

```
请设计一个创意大胆风格的 SaaS 产品落地页，具体要求：

【整体风格】
- 充满创意和个性，打破传统 SaaS 页面的束缚
- 色彩鲜明，视觉冲击力强
- 活泼、有趣、富有人性化
- 适合面向创意工作者或年轻用户群体的产品

【配色方案】
- 大胆的多色组合（不拘泥于传统的单一主色调）
- 主色调：鲜艳的颜色组合，例如：
  - 紫色 (#7c3aed) + 黄色 (#fbbf24) + 青色 (#06b6d4)
  - 粉色 (#ec4899) + 橙色 (#f97316) + 绿色 (#10b981)
  - 红色 (#ef4444) + 蓝色 (#3b82f6) + 黄色 (#f59e0b)
- 背景：浅色柔和背景（奶白色、浅粉、浅蓝等）或白色
- CTA 按钮：使用最鲜艳的颜色，尺寸较大
- 文字：深色（#1f2937）便于阅读

【布局结构】
- Hero 区域：
  - 非对称布局或错位排列
  - 大标题带创意排版（可以倾斜、弯曲、大小不一）
  - 插画或 3D 角色占据显著位置
  - CTA 按钮设计独特（大尺寸、有趣的文案）
  
- 功能介绍：
  - 波浪形分割线或不规则形状
  - 每个功能区域使用不同的背景色块
  - 插画风格的功能图标
  - 错落有致的卡片排列

- 互动区域：
  - 可以加入趣味的微交互
  - 鼠标悬停时有趣的动画反馈
  - 可能包含小游戏或互动元素

- 客户评价：
  - 卡通化的用户头像
  - 对话气泡风格的评价框
  - 彩色的引用标记

【视觉元素】
- 插画风格：
  - 手绘风格的插画
  - 扁平化但色彩丰富的图形
  - 3D 卡通角色
  - 涂鸦风格的装饰元素

- 形状和图案：
  - 不规则的有机形状（blob shapes）
  - 波浪线、锯齿线、手绘线条
  - 彩色的圆点或几何图案背景
  - 手绘箭头和标注

- 图标：
  - 彩色图标（不是单色）
  - 带有阴影和深度的图标
  - 可爱或拟人化的图标设计

- 装饰元素：
  - 涂鸦风格的装饰
  - 贴纸效果的元素
  - 手绘的强调线和圈圈
  - 五彩纸屑效果

【排版】
- 字体：
  - 标题：圆润的无衬线字体（如 Poppins、Fredoka、Quicksand）或手写风格
  - 正文：易读的无衬线字体（如 Inter、DM Sans）
- 排版创意：
  - 文字大小对比强烈
  - 可以使用倾斜、旋转的文字
  - 关键词用不同颜色高亮
  - 手写风格的标注或注释
- 字号：
  - 主标题：超大（72-96px），可以分行错落排列
  - 强调词：特大并使用鲜艳颜色

【动画效果】
- 欢快的动画：
  - 元素弹跳进入（bounce in）
  - 摇摆动画（wiggle）
  - 悬停时的弹性效果（elastic hover）
  - 点击时的庆祝动画（confetti）
  
- 微交互：
  - 按钮按下时的挤压效果
  - 卡片翻转效果
  - 鼠标跟随的趣味元素
  - 加载动画使用有趣的角色或形状

- 滚动动画：
  - 元素从不同方向飞入
  - 视差效果（但要夸张和有趣）
  - 颜色渐变变化

【插画和图形】
- 角色设计：
  - 友好、可爱的卡通角色
  - 多样化的角色（不同肤色、性别、职业）
  - 角色在使用产品的场景

- 场景插画：
  - 工作场景的创意化呈现
  - 夸张的视觉隐喻
  - 充满细节和彩蛋的插画

【按钮和 CTA】
- 大尺寸的 CTA 按钮
- 有趣的按钮文案（如 "Let's do this!" 而不是 "Get Started"）
- 按钮带有阴影、渐变或 3D 效果
- 悬停时有夸张的动画效果
- 可以使用不规则的按钮形状

【响应式设计】
- 移动端：保持趣味性，简化插画
- 确保彩色元素在小屏幕上不会过于拥挤
- 动画效果在移动端适当减少

【语气和文案】
- 友好、轻松、有趣的文案
- 使用表情符号 ✨🎉🚀
- 避免过于正式的商务用语
- 可以使用一些俏皮话或双关语
```

## 🔧 可定制参数

根据具体需求，修改以下参数：

- **色彩方案**：根据品牌选择 2-3 个主色
- **插画风格**：手绘 / 扁平 / 3D / 涂鸦风格
- **活泼程度**：根据目标用户调整趣味性的程度
- **文案语气**：从轻松到俏皮的不同程度
- **目标群体**：创意工作者 / 年轻用户 / 教育 / 娱乐等

## 💡 使用技巧

1. **保持可读性**：虽然创意，但不要牺牲信息的清晰度
2. **品牌一致性**：确保创意风格与品牌个性匹配
3. **性能优化**：大量插画和动画要注意优化加载速度
4. **A/B 测试**：大胆的风格可能有争议，可以测试效果

## ⚠️ 注意事项

- **可访问性**：确保色彩对比度足够，满足 WCAG 标准
- **专业平衡**：即使创意，也要保持一定的专业感
- **目标受众**：确认目标用户能接受这种风格
- **加载性能**：大量插画和动画要做好优化
- **响应式**：在不同设备上测试视觉效果
